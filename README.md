# Frogmouth

A Markdown viewer / browser for your terminal, built with [Textual](https://github.com/Textualize/textual).

Forked from [Textualize/frogmouth](https://github.com/Textualize/frogmouth).

## Installing

Install [mise](https://mise.jdx.dev/) and [uv](https://docs.astral.sh/uv/), then:

```bash
mise use -g uv
uv tool install frogmouth
```

## Usage

```bash
frogmouth README.md
frogmouth https://example.com/doc.md
frogmouth gh textualize/textual
```

Press <kbd>F1</kbd> for help inside the app.
